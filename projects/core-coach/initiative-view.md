# Q3 — Initiative Summary: Coach Foundations

Q3 is about making Coach feel like one product across Culture Amp — unified, context-aware, and trustworthy enough to reach 50% of monthly active platform users.

*Generated from quarterly-priorities.md. Last updated: 2026-06-23.*

---

| Initiative | Stream | What it is | Why it matters | Status |
|---|---|---|---|---|
| Unified Sessions | Unified | A single session history and navigation layer across all Coach-enabled pages. | Users currently experience Coach as disconnected per-page tools — unifying sessions makes Coach feel like one product and removes the biggest friction to repeat use. | Committed |
| Routing | Unified | Cross-agent routing so a question in one Coach context can be answered by the right agent. | Removes the seam between Coach contexts so users get the right answer regardless of where they started the conversation. | Committed |
| Enablement | Unified | A playbook, control plane, eval pipeline, and direct support for teams shipping Coach in Goals and Perform. | Without this infrastructure, teams building on Coach have no shared standards or confidence tooling — this makes Coach extensible at scale. | Committed |
| | | | | |
| Customer Knowledge Base | Context-Aware | A CZ admin interface for uploading company-specific context that Coach uses to ground its responses. | Generic responses are the main reason Coach feels unhelpful to specific customers — company context makes Coach meaningfully more relevant. | Committed |
| | | | | |
| MSI Slice 2 | MSI | A set of improvements to Coach's manager survey insight experience: explainability, dynamic prompts, exec summaries, charts, and action-linking. | MSI is Coach's highest-engagement surface — this slice closes the gap between surfacing insights and helping leaders act on them. | Committed |
| | | | | |
| FE Improvements | Trusted & Insightful | A context pillow, unified disclaimer, and secondary nav improvements across Coach surfaces. | Small UX gaps (what data is Coach using? what are its limits?) erode trust — these fixes make Coach feel polished and transparent. | Committed |
| BE Improvements | Trusted & Insightful | Compliance and codebase health work: GDPR automation, OCSF audit log compliance, and a Datadog spike. | Non-negotiable foundations — without these, Coach can't be sold into regulated markets or operated reliably at scale. | Committed |
| Web Sockets Clean Up | Trusted & Insightful | Ensuring web socket support is consistent and correct across all Coach instances. | Inconsistent web socket behaviour is a source of subtle reliability issues that are hard to debug and damage user trust. | Committed |
