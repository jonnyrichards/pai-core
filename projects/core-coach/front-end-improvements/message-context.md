## Overview

Two distinct layers for providing message context:

### 1. Context Pillow (future-focused)
Show main page context in a 'pillow' underneath the Coach input box so the user can glance at page context when messaging Coach. Sets intent/focus for what you're about to discuss.

### 2. Message Metadata (past-tense)
Show page context + date/time on every Coach message after it is sent (like a receipt) so:
1. User can understand the context of older messages
2. Culture Amp can efficiently audit the message trail
