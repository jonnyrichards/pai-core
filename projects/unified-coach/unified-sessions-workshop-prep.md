# Goal

To align on an approach to navigation and session history so that the UX of Coach feels unified across the platform.

Specifically: Create a set of agreed assumptions, things we know to be true, and resulting principles that we can point to when we make decisions as to how this UX will be implemented in code.

- **Align** = we are in agreement on what sort of experience we want to implement
- **UX of Coach** = this is fundamentally about how Coach behaves in the front end; these are questions which go to what it feels like to use Coach
- **Assumptions** = things we don't yet know but assume
- **Know to be true** = truths we can cling to
- **Principles** = principles with broad application that can guide one or many decisions
