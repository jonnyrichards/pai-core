## Writability: Coach writes into the platform via MCP

Enable Coach to take write actions in Culture Amp — not just surface information, but create and update records. MCP as the mechanism.

**Examples:**
- Coach creates a goal
- Coach creates a 1-on-1 topic
- Coach creates a development plan (aligned with Develop's current sprint work)

**Why it matters:** Moves Coach from advisor to actor. Closes the loop between insight and action — the user doesn't have to leave the conversation to do the thing Coach just recommended.

**Coach's role:** Define the principles and operating model for writability — a spec that other teams use as a guide when implementing on their end. Own the rulebook and a hero use case; don't own the implementation. Think: writability principles, guardrails, UX patterns for write actions, consent model.
