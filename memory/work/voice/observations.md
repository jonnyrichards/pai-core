- 2026-03-31: Coach Voice latency — shared with O: as exec note to Priority Weekly. Root cause: chained architecture (STT → Coach → TTS); Coach service is the primary bottleneck, not voice-specific parts. Proposed optimisations: (1) pre-fetch EL tokens on page load (cold start fix); (2) remove timeout mitigations added for mic feedback; (3) move TTS to client-side (remove BFF); (4) swap to EL Flash models (faster + less mawkish emotional tone); (5) swap LLM to Haiku (speed/quality trade-off). Three options: [RECOMMENDED] implement optimisations + roll out to CZ as planned; hold on TTS, release Dictation first; rearchitect to speech-to-speech (faster but loses text-based guardrails). [work]
